# Synapse 🧠

Synapse is a comprehensive, AI-powered academic management and career readiness platform built to serve as the ultimate hub for students. Developed using the MERN stack (MongoDB, Express, React, Node.js), this application seamlessly blends day-to-day academic tracking with powerful, long-term career preparation tools.

At its core, Synapse is designed to consolidate the fractured student experience. Instead of juggling multiple apps for chatting, tracking habits, storing resumes, and managing study schedules, Synapse brings it all into a single, cohesive ecosystem enhanced by intelligent automation.

### Key Features 

* **🧠 RAG-Powered AI Engine:** Implements advanced Retrieval-Augmented Generation (RAG) allowing users to intuitively query their uploaded course materials, syllabi, and notes for instant, context-aware AI assistance.
* **🤖 Advanced Career Vault:** A secure repository for professional documents integrated with Google's Gemini API for intelligent resume parsing, ATS-friendly HTML template generation, and automated recruiter-style analysis.
* **⚡ Real-Time Collaboration:** A built-in messaging system using WebSockets allows students to connect, form study groups, and collaborate effortlessly without ever leaving the platform.
* **📚 Smart Academic Tracking:** Track semester progress, manage assignments, and calculate CGPA. The platform includes an AI-driven study planner that adapts to upcoming exams and daily habits.
* **📊 Placement Readiness Dashboard:** Analyzes students' academic progress and provides honest, stage-aware feedback on their career readiness, highlighting areas for improvement and skill gaps.
* **🔔 Unified Notification Center:** A cross-platform alert system that seamlessly aggregates messages, missed calls, and academic reminders into a dynamic interface.

### Tech Stack 💻
* **Frontend:** React.js, TailwindCSS (Dynamic UI/UX with micro-animations)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Real-time & AI:** Socket.io (WebSockets), Retrieval-Augmented Generation (RAG) Architecture, Google Gemini API
